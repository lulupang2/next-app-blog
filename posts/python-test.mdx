# 요약

## 1. 데이터와 정보

- **데이터**: 관찰이나 측정을 통해 얻은 수치나 문자 등의 질적 또는 양적 값.
- **정보**: 데이터를 처리하여 문제 해결에 도움이 되는 결과물.
- **컴퓨터**: 데이터를 자동으로 처리하는 기계. 입력, 저장, 연산, 제어, 출력 기능을 가짐.
- **디지털 컴퓨터**: 0과 1로 동작. 데이터와 명령문도 0과 1로 표현됨.
- **프로그램**: 컴퓨터가 작업을 자동으로 처리할 수 있도록 작성된 코드. 프로그래머가 작성.
- **알고리즘**: 문제 해결을 위한 단계별 절차.

## 2. 컴퓨터의 구성

- **하드웨어**: 입력장치, 연산장치, 기억장치, 출력장치, 제어장치로 구성.
- **폰 노이만 구조**: 프로그램과 데이터를 메모리에 저장하고 처리.
- **시스템 소프트웨어**: 운영체제, 컴파일러, 유틸리티 등이 포함됨.
- **운영체제**: 하드웨어를 제어하고 사용자와 컴퓨터 간 중계 역할.

## 3. 파이썬 소개

- **개발자**: 히도 판로쉼(1991년).
- **지원 패러다임**: 명령형, 절차적, 객체지향, 함수형 프로그래밍.
- **용도**: 교육, 웹 개발, 데이터 분석 등 다양한 분야.
- **특징**: 플랫폼 독립적, 인터프리터식, 동적 타이핑 언어.

## 4. 파이썬 기본 문법

- **표현식**: 피연산자와 연산자를 이용한 표현식.
- **함수**: 특정 작업을 수행하는 코드 집합.
- **들여쓰기**: 코드 블록을 표현.
- **주석**: 코드 설명을 위해 사용 (#, """ 사용).
- **변수**: 데이터를 임시 저장.
- **식별자**: 프로그램 내 객체의 이름.
- **산술연산자**: 정수 나눗셈(//), 모듈로(%).
- **내장 함수**: 기본 제공 함수.

## 5. 프로그래밍 구조

- **순차 구조**: 명령을 순서대로 실행.
- **선택 구조**: 조건에 따라 명령을 선택적으로 실행.
- **반복 구조**: 특정 조건에 따라 명령을 반복 실행.

## 6. 선택 구조

- **조건**: 참(True)과 거짓(False)으로 결정.
- **불리언식**: 비교연산자를 사용.
- **논리연산자**: 두 논리값을 연산.
- **단락평가**: 첫 논리값만으로 결과를 판단.

## 7. 반복 구조

- **while 문**: 조건이 참인 동안 명령문 반복.
- **리스트**: 값의 집합체 저장.
- **for 문**: 시퀀스의 각 요소를 반복.
- **중첩 반복**: 반복 구조 내 반복 구조 포함.

## 9. 함수

- **반환값**: 반환값이 있는 함수와 없는 함수로 구분.
- **동시 할당**: 여러 변수에 값을 동시에 할당.
- **스코프**: 변수의 참조 영역.
- **기본 매개변수**: 기본값을 가진 매개변수.
- **가변 매개변수**: \*를 사용하여 가변 개수의 매개변수 지정.

## 10. 객체지향 프로그래밍

- **특징**: 추상화, 캡슐화, 상속, 다형성.
- **클래스**: 상태와 연산을 표현하는 단위.
- **메소드**: 객체의 행동 정의.
- **생성자**: 객체 생성 시 호출.
- **데이터 은닉**: 직접 접근을 방지.

## 11. 모듈과 패키지

- **모듈**: 함수, 상수, 클래스의 집합.
- **패키지**: 하위 패키지 및 모듈의 집합.
- **라이브러리**: 패키지 및 모듈의 집합.
- **import 구문**: 모듈을 네임스페이스에 추가.
- **math, random, time 모듈**: 수학 계산, 난수 생성, 시간 처리 기능 제공.

## 12. 파일 처리

- **파일**: 연속된 바이트로 저장된 데이터.
- **파일 작업**: 파일 객체 생성, 읽기/쓰기, 파일 객체 삭제.
- **딕셔너리**: 키-값 쌍을 저장하는 시퀀스.

## 13. 소프트웨어 개발 라이프사이클

- **단계**: 계획, 분석, 설계, 구현, 테스트, 유지보수.
- **배치 프로세스**: 대량 데이터 처리.
- **게임 루프**: 초기화, 이벤트 검사/처리, 업데이트 과정 반복.

## 14. 리스트와 2차원 리스트

- **리스트**: 순서화된 값의 집합체.
- **2차원 리스트**: 행 인덱스와 열 인덱스를 포함하는 구조.
- **순회**: 2개의 반복 구조가 중첩된 형태로 각 차원을 순회.
